Boiler Plate para projetos na arquitetura MVC com Webpack. 
Algumas coisas precisam ser modificadas como a inserção do .env com a string de conexão com o banco de dados de sua escolha.
Neste projeto, a conexão com o banco de dados é estabelecida antes do servidor ser capaz de receber requisições, portando, configurar, de antemão, o banco de dados e sua conexão é essencial.
